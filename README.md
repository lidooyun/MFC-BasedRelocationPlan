# MFC-BasedRelocationPlan
为参加数学建模竞赛，独立开发一套本地 MFC 对话框应用，用于城市搬迁方案的可视化与优化。用户可导入城区平面地图，手动画框/画线标注房屋与街道；程序基于遗传算法对标注区域进行多目标优化（成本最小、满意度最大），并实时填充色块展示最优搬迁方案，一键导出高清结果图，辅助论文配图与决策分析。 技术栈：C++、MFC、GDI+、遗传算法（自写）、离线地图位图标注、csv配置文件

## 可执行文件使用方式
运行环境要求：
1.Windows操作系统
2.安装了python并且正确设置了环境变量（开发版本是Python 3.12.10）若已安装Python且正确配置环境变量后，仍报错未找到对应的包，请按照本文档最末说明依次执行命令行语句下载安装对应的包
3.软件当前目录到本盘符根目录的整个路径中，文件夹名不能有中文和特殊字符

软件运行所需的py包：pandas matplotlib numpy scipy csv random time collections 
****************************************
第一步：数据化
1)对于仅有俯瞰图/区域示意图的情形，运行程序后点击“导入图片”按钮，在导入图片后需手动标注院落与街道。院落标注时，单选框勾选“院落标注”后，用鼠标在窗口内绘制即可；街道标注时，单选框勾选“街道标注”后，用鼠标在窗口内绘制即可；勾选“指针”释放画笔。

2)对于有各院落与街道测量数据的情形，需要准备两份符合“测量数据导入格式”[1]的csv文件，运行程序后依次点击“导入院落数据”“导入街道数据”“生成标注图”按钮即可在窗口内显示出数字化信息。

第二步：运算各户与街道间的最近距离
在完成第一步，内存中写入了院落与街道数据后，依次点击按钮“计算与街距离”“导出与街距离”。程序会在工作路径下生成数据文件“院落离街道的距离.csv”。

第三步：获得各户搬迁意愿去向
确保工作路径中有文件“附件一：老城街区地块信息.csv”[2]，“第一问代码.py”后，点击按钮“生成搬迁方案”。程序会在工作路径下生成数据文件“问题一最终搬迁方案表-提供修缮补偿.csv”。

第四步：获取整院面积最大搬迁方案
确保工作路径中有文件“adjoin.csv”[3]，“问题一最终搬迁方案表-提供修缮补偿.csv”，“附件一：老城街区地块信息.csv”，“第二问代码.py”后，点击按钮“腾空搬迁方案”。程序会在工作路径下生成该方案文件“问题二搬迁方案.csv”。
通过按钮“搬前分布示意”“搬后分布示意”，可在窗口内查看搬迁前后居民居住变化情况。

第五步：获取性价增益最高的搬迁方案
确保工作路径中有文件“adjoin.csv”[3]，“问题一最终搬迁方案表-提供修缮补偿.csv”，“附件一：老城街区地块信息.csv”，“第三问代码.py”后，点击按钮“最优搬迁方案”。程序会在工作路径下生成该方案文件“问题三搬迁方案.csv”。
通过按钮“搬前分布示意”“搬后分布示意”，可在窗口内查看搬迁前后居民居住变化情况。
******************************************
[1]测量数据导入格式：院落信息数据要求归一化至1500*720像素坐标系（不同显示器可能有细微差异）中，包含院落中心点的横纵坐标，院落的近似长与近似宽，保存在文件“院落位置信息.csv”中，具有文件头[x,y,len,wid]，放置于与可执行文件“平移置换.exe”同目录下。街道信息数据要求归一化至1500*720像素坐标系中，包含街道起始点横纵坐标，街道终止点横纵坐标，保存在文件“街道位置信息.csv”中，具有文件头[x1,y1,x2,y2]，放置于与可执行文件“平移置换.exe”同目录下。

[2]老城街区地块信息.csv格式：文件中有六列，依次是地块ID，院落ID，地块面积，院落面积，地块方位，是否有住户。具有文件头[PlotID,YardID,PlotArea,YardArea,PlotDirection,IsInhabited]。文件中不能有中文，方位使用[East,West,South,North]代替“东西南北”。

[3]adjoin.csv院落毗邻情况表格式：需人工根据俯瞰图或实际测量数据，判断每个院落与之毗邻的院落编号并填写该表格。数据格式要求：文件中有不定列，其第一列为每个院落编号，第二列及之后均为与之毗邻的院落编号，具有文件头[Id,AdjoinId1,AdjoinId2,AdjoinId3...]。
